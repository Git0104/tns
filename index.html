About TanPvP

🌊 AFK Pool System

特定の水の中に滞在することで、15分ごとに1 Killを獲得できます。放置している間もスコアを伸ばしましょう。

⚔️ Custom Kits

Default、VIP、VIP+など、多彩なキットを用意。各プレイスタイルに合わせた装備で戦場を支配せよ。

🛡️ Block Protection

自分が設置したブロックのみ破壊可能。マップの美しさを保ちつつ、戦略的な建築が可能です。

Server Rules

  • 禁止 チートツールの使用 (Killaura, Fly, etc...)
  • 禁止 意図的なラグの発生、サーバー負荷行為
  • 禁止 他プレイヤーへの過度な暴言、嫌がらせ
  • 許可 自分が設置したブロックの破壊
  • 重要 AFKプールでの放置（推奨されています！）
